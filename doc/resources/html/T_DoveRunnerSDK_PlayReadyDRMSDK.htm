      PlayReadyDRMSDK Class

      A class that can create DownloadTask and acquire PlayReady license. To use PlayReadyDRMSDK, you need to set the MediaProtectionManager in the media player through the CreateProtectionManager function. And you can get the DownloadTask object through the createDownloadTask method to download the content.

      ConfigureHardwareDRM This method will configure the properties used by Media Foundation and PlayReady to specify Hardware DRM. The existing ProtectionManager assigned to the MediaElement is altered with updated/removed properties. Lastly, proactive individualiation is called. You must configure the MediaProtectionManager through CreateProtectionManager before calling this function.
      ConfigureSoftwareDRM This method will configure the properties used by Media Foundation and PlayReady to specify Software DRM. The existing ProtectionManager assigned to the MediaElement is altered with updated/removed properties. Lastly, proactive individualiation is called. You must configure the MediaProtectionManager through CreateProtectionManager before calling this function.
      CreateDownloadTask This task can be used to download remote content to local storage. Downloaded files are stored in the Local(LocalState) Storage. You can also check download status and update UI for progress by using Handler.
