      DoveRunnerSDK.DownloadTask Namespace

      Namespace for using the DoveRunnerDownloadTask class

      Classes

      DownloadTask A class that receives remote URL and downloads them locally. You can create the instance by using PlayReadyDRMSDK.createDownloadTask function. You can also cancel current download using the same instance.

      Delegates

      DownloadCompleteHandler Handler to notify download complete
      DownloadFailHandler Handler to notify download failure
      DownloadProgressChangedHandler Handler to notify download progress changed
      DownloadSendRequest Handler to callback download request
